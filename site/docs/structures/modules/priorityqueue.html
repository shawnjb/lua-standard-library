Module priorityqueue

Priority queue is a data structure that allows to insert elements with a priority and then retrieve them in order of priority.

Functions

new () Creates a new priority queue.
insert (value, priority) Inserts a new element in the priority queue.
pop () Retrieves the element with the highest priority.
peek () Retrieves the element with the highest priority without removing it.
count () Retrieves the number of elements in the priority queue.
isEmpty () Checks if the priority queue is empty.
heapifyUp (index) Heapifies the priority queue up.
heapifyDown (index) Heapifies the priority queue down.
swap (index1, index2) Swaps two elements in the priority queue.
